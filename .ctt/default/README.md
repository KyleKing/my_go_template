## Development

### Prerequisites

- Go (see `go.mod` for version)
- [mise](https://mise.jdx.dev/) for task running
- [hk](https://hk.jdx.dev/) for git hooks

### Setup

```bash
# Install tools
mise install

# Install git hooks
hk install --mise

# Run tests
mise run test

# Run all checks
mise run ci
```

### Testing

```bash
# Run tests with coverage
mise run test

# View coverage report
mise run test:view-coverage
```

### Formatting

```bash
# Auto-fix linting issues
mise run format

# Or run hooks manually
hk fix
```

## Maintenance

### Updating Dependencies

go get -u ./... && go mod tidy && go test ./...
